# Requirements
- Node.js (v16.17.0)
- NPM (v9.6.4)
- PostgreSQL (v15.2)

# Installation and Setup

- Clone the repo
- Issue `npm install` command
- Create `.env` from `.env.sample` and update environment configurations

# Build and Run

- `npm run start`

# Tests and Lint
 ```sh
  # Run all unit tests
  npm run test

  # Run lint checks
  npm run lint

  ```
